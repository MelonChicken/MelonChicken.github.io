# ArchiTag – 대통령기록관 시청각기록 메타데이터 자동 생성 시스템

---

## 🎯 Executive Summary

- **문제:**
  대통령 시청각기록물은 관리용 메타데이터는 존재하지만, **내용 기반 메타데이터\(Description, Subject\)는 선택 요소이며 수작업에 의존**하여 실제로는 공백 상태가 발생한다

- **해결:**
  VLM을 활용하여 이미지로부터 **Description\(내용 기술\)과 행사 카테고리**를 자동 생성하고,
  
  이를 기존 메타데이터와 비교하여 **AI 보조\(HITL\) 기록관리 워크플로우의 실효성**을 검증한다

- **결과 \(목표\):**
  - Description 품질 \(BERTScore, ROUGE\)
  
  - 행사 분류 성능 \(Macro F1\)
  
  - 처리시간 및 재작업률 변화

- **다음 단계:**
  - 데이터 수집 및 어노테이션 기준 정의
  
  - VLM 기반 생성 실험
  
  - HITL 워크플로우 비교 분석
