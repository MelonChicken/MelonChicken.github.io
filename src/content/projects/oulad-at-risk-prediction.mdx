## Problem

VLE에서 수집하는 학생 데이터를 토대로 학습 어려움이 있는 학생을 조기 탐지

## Motivation

Behavioral prediction from interaction logs — the 'behavior as signal' idea applied to learners.

## Method

Leak-free active-at-cutoff cohort; behavioral + demographic features; classifiers @ weeks 5/7/10; benchmarked vs Staneviciene et al. (2024).

## System / Pipeline

VLE logs → leakage-controlled features → models @ W5/W7/W10 → evaluation

## Result

Best F1 0.593 (DecisionTree, W7); best AUC 0.739 (NaiveBayes, W10).

## Limitation

Class imbalance; snapshot features plateau without richer temporal signal.

## Next Step

Reframe as a temporal sequence problem rather than snapshot features.
