## Problem

VLM을 활용해 기록물의 내용 기반 메타데이터를 자동 생성하고, 아카이브 검색성과 운영 효율을 개선하는 연구

## Motivation

A direct test of VLM-assisted labeling — the core method for the labeling-automation program.

## Method

VLM generates content metadata; retrieval + operations flow designed around it (BPMN).

## System / Pipeline

image → VLM caption/tags → metadata index → search
