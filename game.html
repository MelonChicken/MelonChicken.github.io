Pose Detection

Rule-Based No ML Model 21 Landmarks Geometry Only

Each finger is classified extended/curled by comparing tip-to-wrist distance vs. PIP-to-wrist distance — rotation-invariant geometry. Thumb uses normalized y-axis vs. palm centroid. Same principle as pose-based action recognition in animal behavior studies.
